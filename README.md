# 五張撲克牌遊戲 (Five-Card Poker)

## 專案簡介
本專案為視窗程式設計 (II) 的課堂作業。這是一個使用 C# 與 Windows Forms (WinForms) 開發的五張撲克牌遊戲。玩家可以在遊戲中進行押注、發牌、選擇換牌，系統會自動判斷最終牌型並依據賠率表計算中獎金額。

- **開發者**：資訊工程學系 2A 藍奕 (學號：1131437)
- **開發環境**：Visual Studio (C# WinForms)

## 功能說明
本系統實作了以下核心功能：
1. **下注系統**：玩家擁有初始資金 (1,000,000)，每局開始前需輸入押注金額，系統會自動扣除本金。
2. **發牌與換牌**：使用亂數洗牌後發出 5 張牌，玩家可點擊牌面選擇保留或更換（最多換一次）。
3. **牌型判斷**：系統能精準判斷同花大順、鐵支、葫蘆、順子等各種牌型。
4. **獎金結算**：依照作業規定的賠率表（如：同花大順 250 倍、鐵支 25 倍等），自動計算並將中獎金額加回總資金。
5. **隱藏測試按鍵**：支援鍵盤快捷鍵 (q, w, e, r, t, y) 快速生成同花大順、鐵支等特定牌型以供測試。

## 執行說明
1. 將本專案 Clone 或下載至本機端。
2. 使用 Visual Studio 開啟專案資料夾中的 `.sln` 方案檔。
3. 點擊上方的「開始」按鈕編譯並執行程式。
4. 於「押注金額」欄位輸入數字並點擊「押注」按鈕。
5. 點擊「發牌」，接著點選想更換的撲克牌，選定後點擊「換牌」。
6. 點擊「判斷牌型」查看結算結果。
